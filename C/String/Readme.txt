The directory has many C file about the deal of string.

bandwidth.c
在做项目miniupnpd时，为了对字符串的初步了解，写了这个测试文件。

char_pointer.c
在C语言中，char类型的指针是非常重要的一种工具。

elf_hash.c
在模块datalib中的一个函数，可以将任何字符串变成一个hash地址（类似于md5值）。

getopt.c getopt_long.c
编译形成一个新的命令：getopt -p/--show

header_struct.c
结构体指针，以及内存中的调用。

magic.c
在模块datalib中的一个函数，采取二进制位移的算法。
第一次：明文（加密）―>暗文
第二次：暗文（解密）―>明文

substr_add.c substr_del.c
删除/添加子串
